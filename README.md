### 项目概述
这是一个仿电商网站的前端项目，主题为"小兔鲜儿"，但实际上是一个精灵宠物交易平台，融合了多个游戏元素（洛克王国、赛尔号、原神、鸣潮等）。

### 主要功能模块 🎯 核心功能
- 轮播图展示 ：自动播放的轮播图，支持手动切换和圆点导航
- 商品分类展示 ：按精灵等级和类型分类展示
- 搜索功能 ：顶部搜索栏
- 购物车 ：显示购物车数量
- 响应式布局 ：适配不同屏幕尺寸 🎮 特色内容
- 多游戏融合 ：包含洛克王国、赛尔号、原神、鸣潮等游戏角色
- 精灵分类 ：新鲜精灵（低等）、人气推荐（高等）、赛尔号精灵等
- 道具售卖 ：咕噜球等游戏道具
### 技术栈
- 前端框架 ：纯 HTML5 + CSS3 + JavaScript
- 图标库 ：iconfont 字体图标
- 样式架构 ：模块化 CSS 设计
- 交互功能 ：原生 JavaScript 实现轮播图
### 设计特点
1. 现代化界面 ：简洁的卡片式布局，清晰的视觉层次
2. 游戏化元素 ：融合多个热门游戏IP，增加趣味性
3. 用户体验 ：
   - 轮播图自动播放，鼠标悬停暂停
   - 分类导航和快速筛选
   - 价格标签和商品信息清晰展示
### 特色亮点
- 跨游戏整合 ：将不同游戏的精灵角色整合到一个平台
- 价格体系 ：从 [ o bj ec tO bj ec t ] 0.99 到 9999.99的完整价格区间
- 互动功能 ：轮播图、弹幕等交互元素
- 幽默元素 ：页面包含一些有趣的注释和彩蛋
